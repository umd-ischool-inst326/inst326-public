== Overview

An introduction to software testing, Python's module import system (including how to import your own modules), and the basics of version control using git.

=== Links
* link:slides.html[Slides]
* link:exercises/index.html[Exercises]

== Learning Outcomes

After completing this module, students should understand:

* Why software testing is important.
* How to organized code for testing.
* How to write tests and run them with pytest.
* How module imports work.
* The basics of version control with git and Github.
