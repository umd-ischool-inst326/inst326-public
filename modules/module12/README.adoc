== Overview
This module introduces students to importing structured data on the web. We learn to use
the requests module, the basics of HTTP, common web data formats including XML and JSON.

== Learning Outcomes

.After completing this module, students should be able to:
* Recognize various web data formats, including XML and JSON
* Parse formatted data using appropriate python modules
* Import data from web formats into python data structures
